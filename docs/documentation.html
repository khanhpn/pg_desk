PgDesk Documentation

Use this guide to install PgDesk, connect to PostgreSQL, browse database objects, and run your first query.

Install Connect Explore Run SQL Troubleshooting
Step 1

Install PgDesk

Download the latest release for your operating system from the GitHub releases page. PgDesk supports macOS, Windows, and Linux builds.

Open latest release
Step 2

Create a PostgreSQL connection

  1. Open PgDesk and select the connection settings action.
  2. Enter host, port, database, username, and password.
  3. Enable SSL if your database requires an encrypted connection.
  4. Test the connection, then connect when the profile is valid.
Step 3

Browse schemas, tables, and views

After connecting, the explorer sidebar loads database metadata. Use the refresh action when database objects change while PgDesk is open.

Select a table or view from the sidebar to open a quick query for that relation.
Step 4

Write and run SQL

Use the SQL editor for ad hoc PostgreSQL queries. Run the query from the toolbar and review rows, empty results, or error messages in the result panel. 

select *
from public.users
limit 100;
Help

Troubleshooting

Connection test fails

Confirm the host, port, database, username, password, and SSL mode match the PostgreSQL server configuration.

Explorer is empty

Refresh the explorer after connecting and confirm the database user has permission to read schema metadata.

Query returns an error

Check the result panel message, then verify table names, schema names, and SQL syntax for the active PostgreSQL database.
